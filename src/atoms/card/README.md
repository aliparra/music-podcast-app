# music-podcast-app

## Card

- A basic card that displays an image (with url and description), a title and an author.

### Example

```tsx
<Card
	image={{
		url: 'https://via.placeholder.com/150',
		description: 'Image description',
	}}
	title="Title example"
	author="Author name"
/>
```
