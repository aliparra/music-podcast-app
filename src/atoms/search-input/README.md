# music-podcast-app

## Search input

- Search input allow users to input custom text to filter results on a list

### Example

```tsx
const handleFilterList = (searchTerm) => {
  console.log(searchTerm)
}

<SearchInput
  filterData={handleFilterList}
  placeholder="Filter elements..."
/>
```
